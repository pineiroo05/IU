Información sobre las clases de validación

Clase: ValidateFieldsForm

Descripción:

Método: crearCampo(id, valor, infoAtributo)

Descripción: Crea un elemento HTML según las normas de la estructura de la entidad, configura sus propiedades, le asigna el valor de prueba y lo añade al DOM.

ParámetroTipoDescripción
idStringIdentificador del nuevo nodo HTML.
valorString/ObjectDato de prueba a introducir en el campo (puede ser un objeto para simular ficheros).
infoAtributoObject

Información devuelta: HTMLElement-El nodo del elemento de formulario recién creado y colocado en el DOM.

Método: validarCampo(validacion, id, parametros)

Descripción: Invoca al metodo de validacion concreto de cada campo.

ParámetroTipoDescripción
validacionStringNombre del método a ejecutar.
idStringID del elemento HTML temporal que contiene el valor que se va a evaluar.
parametrosMixedCriterios de validacion de la regla.

Información devuelta: Boolean/String-Devuelve true si pasa la prueba. Si falla, devuelve el codigo de error. Si la regla existe, devuelve un mensaje de aviso.

Método: simulacionFichero(campo, objeto)

Descripción: crea un fichero simulado para el testeo de los campos relacionados con los ficheros.

ParámetroTipoDescripción
campoHTMLElementEl nodo input file donde se monta el archivo.
objetoObjectElemento donde se describen las caracteristicas del fichero a crear (tamaño,tipo,nombre...).

Información devuelta:void

Método: limpiar()

Descripción: Limipia por completo el contenedor temporal eliminando todos los elementos HTML que se añadieron previamente.

Parámetros: Ninguno.

Información devuelta:void

Clase: Validations

Descripcion:

Metodo: min_size(id,minimo)

Descripcion: Metodo para comprobar si el tamaño del campo considerado es mayor del minimo requerido.

ParametrosTipoDescripcion
idStringIdentifica el campo sobre el que se van a hacer las pruebas
minimoIntegerValor utiizado para comparar con el tamaño del valor asignado al campo. Representa el minimo aceptado.

Informacion devuelta: boolean, true si el valor esta por encima del minimo y false en otro caso.

Metodo: max_size(id,maximo)

Descripcion: Metodo para comprobar si el tamaño del campo considerado es menor del maximo requerido.

ParametrosTipoDescripcion
idStringIdentifica el campo sobre el que se van a hacer las pruebas
maximoIntegerValor utiizado para comparar con el tamaño del valor asignado al campo. Representa el maximo aceptado.

Informacion devuelta: boolean, true si el valor esta por debajo del maximo y false en otro caso.

Metodo: exp_reg(id,expReg)

Descripcion: Metodo para comprobar si el valor del campo cumple con el formato requerido.

ParametrosTipoDescripcion
idStringIdentifica el campo sobre el que se van a hacer las pruebas
expRegStringExpresion utilizada para ver si el valor del campo cumple con el formato requerido.

Informacion devuelta: boolean, true si el valor cumple el formato y false en otro caso.

Metodo: not_exist_file(id)

Descripcion: Metodo para comprobar si hay o no un fichero subido al campo que lo requiera.

ParametrosTipoDescripcion
idStringIdentifica el campo sobre el que se van a hacer las pruebas

Informacion devuelta: boolean, true si existe fichero y false en otro caso.

Metodo: max_size_file(id,tamMax)

Descripcion: Metodo para comprobar que el tamaño del fichero no exceda un valor maximo.

ParametrosTipoDescripcion
idStringIdentifica el campo sobre el que se van a hacer las pruebas
tamMaxIntegerValor utiizado para comparar con el tamaño del fichero este por debajo del maximo aceptado.

Informacion devuelta: boolean, true si el valor esta por debajo del maximo y false en otro caso.

Metodo: type_file(id,arrayTipos)

Descripcion: Metodo para comprobar si el tipo del fichero subido, se corresponde con alguno de los tipos aceptados.

ParametrosTipoDescripcion
idStringIdentifica el campo sobre el que se van a hacer las pruebas
arrayTiposObject???Array que contiene los tipos de fichero con los que se compara el tipo del fichero subido.

Informacion devuelta: boolean, true si el valor esta por encima del minimo y false en otro caso.

Metodo: format_name_file(id,expReg)

Descripcion: Metodo para comprobar si el valor del nombre del fichero cumple con el formato requerido.

ParametrosTipoDescripcion
idStringIdentifica el campo sobre el que se van a hacer las pruebas
expRegStringExpresion utilizada para ver si el valor del nombre del fichero cumple con el formato requerido.

Informacion devuelta: boolean, true si el valor cumple el formato y false en otro caso.

Metodo: min_file_name_size(id,minimo)

Descripcion: Metodo para comprobar si el tamaño del nombre del fichero es mayor que el minimo requerido.

ParametrosTipoDescripcion

minimoIntegerValor utiizado para comparar con el tamaño del nombre del fichero. Representa el minimo aceptado.

Metodo: max_file_name_size(id,maximo)
